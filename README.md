# Fuzzy logic framework

### Minimum configuration required :
- cmake version 3.9.6 or more
- g++ GNU compiler
- QT library version 5


#### Build project :

```BATCH
$ git clone https://github.com/LaroyenneG/fuzzy-logic.git
$ cd fuzzy-logic
$ cmake CMakeLists.txt
$ make
```
